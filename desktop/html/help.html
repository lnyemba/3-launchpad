Features:
  • Plays music (mp3)
  • Continuous access to your library
  • Desktop & mobile editions
  • Free custom streaming
  • Sharing via cloud (dropbox; google)
  • Creates & Stores Playlists
Road map (next release)
  • Full dropbox integration
  • Sugarsync integration
  • Public Internet music
  • Better user experience and a lot more
