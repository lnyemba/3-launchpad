What is 3-launchpad?
3-launchpad (or e-launchpad) is a cloud enabled media player that aggregates music from various storage providers like google; and dropbox. As we develop we will integrate other storage providers and deliver a uniquely rich media experience. To learn more visit us at http://www.3-launchpad.org

Features
  • Rich & unique theme engine
  • Cloud storage music integration: dropbox; google docs
  • Local hard drive music integration
  • Absolutely cross-platform (Android; Linux; Mac; and Windows)
  • Absolutely free & advertisement free!!
